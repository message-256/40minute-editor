a small editor i made becuase i felt like it\
usage:\
n+a+b: append to index n the character b(cant remember if i implemented it right)\
n+s+b: replace index n with the character b\
n+d: delete index n\
\>text/replacement: replace text with replacement accross the entire file (this is one i know works(if weirdly))\
w+file.name: write buffer to file.name\
r+file.name: read file.name into the buffer\
p: print buffer\
q: exit\
w: write contents to last file written or read
